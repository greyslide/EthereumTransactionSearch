# EthereumTransactionSearch

Created with .NET 5.0 (Core) Web API and Angular 8 with Angular CLI V6.0.0

Split into N-Tier architecture with API, application services, UI (Angular), Domain and DataObjects layers.

## Development environment setup

- Clone repo
- Open EthereumTransactionSearch.sln file in VS2019 and build solution
- Install node.js and add to system environment variables path if not already there
- Change directory in console as administrator to EthereumTransactionSearch.UI/ClientApp and run 'npm install'

## Running locally

- Run EthereumTransactionSearch.API in VS2019 in IIS Express
